## 💻 Git Commands Used

### Initialize Repository

```bash
git init
```

### Add Files

```bash
git add .
```

### Commit Changes

```bash
git commit -m "Initial commit"
```

### Verify Repository Status

```bash
git status
```

### List Available Branches

```bash
git branch
```
