# 📘 Git Hands-On Lab 1

## 🎯 Objective
Learn the basic Git workflow by creating a local repository, tracking a file, and making the first commit.

---

## 🛠️ Tools Used
- Git
- Git Bash
- Windows

---

## 📂 Files Included
- 📄 `welcome.txt` – Sample text file
- 🖼️ `Output.png` – Screenshot of the Git commands and output
- 📄 `README.md` – Project documentation

---

## ✅ Commands Executed

```bash
pwd
git init
git status
git add welcome.txt
git status
git commit -m "Initial commit"
```

---

## 📌 Outcome
- ✔️ Initialized a local Git repository.
- ✔️ Added a new file (`welcome.txt`) to the staging area.
- ✔️ Created the first commit successfully.
- ✔️ Verified the Git workflow using Git Bash.
